# ToolBox server  Challenge

## Installation

### local dev

```bash
npm install
```
### run with nodemon
```bash
npm run start-dev
```
### run with node 
```bash
npm run start
```
### with Docker 

```bash
docker build . -p 3000:3000 -t toolbox-server:lastet
```
